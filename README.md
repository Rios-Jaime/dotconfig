# dotconfig
This repo has my config files for use in other machines


How to use:
1. Clone repo
```
    n -s /path/to/original /path/to/link
```
3. add submodules
```
    git submodule init
    git submodule update
    git submodule sync
```
4. Fire up vim and install plugins using vundle
    :PluginInstall
